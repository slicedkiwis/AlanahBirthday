## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

4. Deploy to GitHub Pages:
```bash
npm run deploy
```

## Deployment to GitHub Pages

1. Update the `homepage` field in `package.json` with your GitHub username and repository name
2. Create a new repository on GitHub
3. Push your code to the repository
4. Run `npm run deploy` to deploy to GitHub Pages

## Structure

- `/src/components/` - React components
- `/src/utils/` - Utility functions and data
- `/public/static/` - Images and audio files
